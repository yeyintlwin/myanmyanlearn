# 2-1 　コンピュータの特徴と種類

コンピューターの基本的な特徴と種類について、またコンピューターの様々な書類について学習します。

---

> コンピュータの定義と特徴

## (1) コンピュータの定義

コンピュータは以下のように定義されます。
① 電気的な処理を行う（機械的に動作するところがない）。
② 動作原理に 2 進数を使っている。
③ 再生可能な記憶装置をもつ。
④ 論理演算回路で構成されている。

## (2 )コンピュータの特徴

コンピュータには以下のような特徴があります。

|        |                                                        |
| :----- | :----------------------------------------------------- |
| 確実性 | 指示した手順どおりの作業をすることができる。           |
| 正確性 | 記録した内容を正確に保持できる。                       |
| 汎用性 | プログラムを変えることで、違う作業をすることができる。 |
| 拡張性 | パソコンに容易に機器の追加ができる。                   |
| 高速性 | 高速に作業を処理する。                                 |

---

> コンピュータの種類

## (1) パーソナルコンピュータ

主に個人で使用するために作られた私たちにもっとも身近なコンピュータです。「パソコン」、「PC（ピーシー）」と略されます。10 年くらい前の大型コンピュータに匹敵する性能のものもあります。

## (2)サーバ

利用者（クライアント）として使用するものに対する供給者（サーバ）としての役割をもったコンピュータのことです。最近では安定稼働、高信頼性などを目的に、RAID の導入、複数 CPU 構成などの工夫を凝らしたものが、サーバ機として販売されています。

## (3)ワークステーション

もともとは CAD や CG 用に開発され、科学技術計算、CAD、グラフィックデザイン、事務処理等に特化した業務用の高性能なコンピュータのことをいいます。

## (4)マイクロコンピュータ

「超小型のコンピュータ」という意味で、特定機能専用のコンピュータとして使われます。電化製品などの制御をするために組み込まれ、直接目にすることはほとんどありません。

## (5) メインフレーム（大型汎用コンピュータ）

航空機の運航管理や座席予約など、私たちの生活に密着したところで使われています。科学技術計算、事務処理、制御用などソフトウェアにより汎用性があります。高価ですが、処理能力と安定性が高いコンピュータです。

## (6) スーパーコンピュータ

計算性能を極限まで高めたコンピュータのことで、複雑な方程式を解くような計算を得意としています。気象情報の解析に使われるなど、天気予報の大きな力にもなっています。
