# 第2章 パソコンの基礎

## 2-5　ソフトウェアの種類と役割

ハードウェアであるコンピュータを利用するために必要なソフトウェアの種類と役割について学習します。

>**● ソフトウェアの役割と分類**

ソフトウェアは、パソコンを利用するために必要なプログラムで構成されており、プログラムの役割により分類することができます。ソフトウェアは、基本ソフトウェアとミドルウェア、アプリケーションソフトに分類されます。

>[ソフトウェア] ──── [基本ソフトウェア]
>
>                   ├── [ミドルウェア]
>
 >                   └── [アプリケーションソフト]

### (1) 基本ソフトウェア

基本ソフトウェアは、ハードウェアの機能を効率的に活用し、ハードウェアとアプリケーションソフトの作業が効率よく行えるように仲立ちをします。
この基本ソフトウェアとOS上で動作し、アプリケーションソフトに対してOSよりも具体的な機能を提供するミドルウェアを総称しシステムソフトウェアといいます。

### (2) ミドルウェア

ミドルウェアは、ハードウェアやOSとアプリケーションソフトとの間に位置するソフトウェアです。複数のアプリケーションで使用する共通機能の提供や、ハードウェアやOSの差異を吸収する機能などをもっています。

### (3) アプリケーションソフト

アプリケーションソフトは、**応用ソフトウェア**ともいい、ワープロや表計算、ゲームなどパソコンを利用する目的ごとにさまざまなソフトウェアが存在します。

>OS (Operating System) : オペレーティングシステム
>
>ミドルウェアには、日本語入カソフトや通信ソフトがあります。
---
>●プログラム言語

パソコンに何かを命令するときは、パソコンが理解して実行するための言語を使う必要があります。このパソコンのための言語をブログラム言語といいます。

## (1) 低水準言語

パソコンが理解可能な命令を**機械語**といい、2進数で表現されます。ただし、このままでは処理の内容を人間が理解することが難しいため、**アセンブラ言語**などの機械話に近い言話を用います。これらを低水準言語といい、次のような特徴があります。
●命令がすべて記号化される。
●アセンブラ言語と機械語は、一対一に対応する。
●データが格納されている場所をアドレスで指定する。
●アセンブラ言語で書かれたプログラムは、アセンブラによって機械語に翻訳し実行する。

## (2) 高水準言語

人間が理解しやすいように処理を表現したプログラム言語のことを高水準言語といいます。

| 言語        | 特徴・用途 |
|-------------|------------|
| Fortran     | 科学技術計算を目的とした言語であり、最初に開発された高水準言語。 |
| COBOL       | 事務処理を目的とした言語 |
| C           | システム開発に用いられる代表的な言語 |
| C++         | Cを基本にオブジェクト指向を取り入れた言語 |
| C#          | マイクロソフトが開発したオブジェクト指向言語 |
| Java        | 様々なOSで動作するアプリケーション開発などに活用されるオブジェクト指向言語 |
| Python      | 機械学習などにも活用される汎用性の高いインタプリタ型言語 |
| Javascript  | 動的サイトを構築する際などに活用されるオブジェクト指向言語 |
| SGML        | 文書構造を定義し、電子化を行ためのマークアップ言語 |
| HTML        | Webページの構築などに活用されるマークアップ言語 |
| XML         | SGMLを基本に使いやすいように簡略化した言語、独自のタグを定義して利用できる。 |

### ① コンパイラ型言語

コンパイラ型言語は、作成したプログラムをあらかじめ実行可能な機械語に翻訳する必要のあるプログラム言語です。

### ②インタプリタ型言語

インタプリタ型言譜は、1行ずつプログラムを読み込んで機横語に翻訳しながら実行するプログラム言語です。
