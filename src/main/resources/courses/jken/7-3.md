# 7-3 　ネチケット （2）

ネットワーク上に情報を発信する手段として、新しい手段が生まれています。それとともに、新しい問題も起こっています。

---

> インターネットでの危険性

インターネット上の情報は、世界中の人が見ることができます。安易に個人情報を発信して、思いもよらない事態に遭遇する可能性もあります。最近では、悪意で個人情報を収集して、金銭を請求したり、迷惑メールを送ったり、ダイレクトメールや勧誘の電話をしたりするケースが多く見られます。
日頃から個人情報の取り扱いに注意し、怪しい Web サイトにアクセスせず、簡単に自分の情報を登録することはしないようにしましょう。もし、迷惑なメールが来ても無視し、脅されても金銭の請求に応じないことが大切です。インターネット上の情報を受け取る場合は、受信する側が、情報の信ぴょう性を自分自身で判断することが必要になります。その他、困ったことが起きたら、警察や消費者生活支援センターに相談しましょう。

---

> SNS やブログのネチケット

① 他人への批判や悪口を載せない。
② 個人情報を載せない。
③ 著作権や肖像権などの権利を侵害しない。
④ 知り合いの人も見る可能性があるので、身辺に被害が及ばないように気を付ける。自分や相手の居場所や行動範囲を安易に公開しない。

---

> Web サイト作成のネチケット

① 許可なく個人情報を載せない
② 著作権や肖像権などの権利の侵害がないようにする。
③ 違法や有害な Web サイトにリンクを貼らない。
④ Web サイトのデザインが、不快感を与えるもるものにならないようにする。
·インターネットオークションのネチケット bq
① 正確な商品情報を載せる。
② 物品や金銭をだまし取る行為をしない。
③ 商品の送付や、金銭の送付をできるだけ早く行う。
④ 商品や金銭を受け取った確認のメールを送る。
⑤ コピー商品を取り扱わない。

---

> スマートフォンや携帯電話のマナー

① 電車やバスでは、優先席付近では電源を切り、他の場所はマナーモードにし、通話は控える。
② 自転車や自動車運転中に通話やメールを使用をしない。
③ 映画館や美術館などでは電源を切る。
④ 飛行機や病院内などの使用を禁じられている所では電切る。
⑤ 入学試験や検定などのテスト中には電源を切る。
⑥ 静かな場所では、声の大きさを抑える。
⑦ カメラ付き携帯電話で撮影する際は、肖像権などにに配る。
⑧ 本屋などで商品を買わずに、勝手に撮影しない。
⑨ 個人として楽しむ目的であっても、撮影を制限して合は撮影しない。
⑩ 商店などで勝手に充電しない。
