## <設間2>次の記録容量に関する記述中のに入れるべき適切な字句を解答群から選べ。
ある新聞では、1面(ベージ)当たり平均4万字の記事が書かれている。4G(ギガ)バイトのUSBメモリには、この新聞<span style="border:1px solid #000; padding: 0 1em 0 1em ">5</span> 面分を記録できる。ただし、文字コードには2バイトの漢字コ-ド (JISX0208)を使用し、写真など画像データは含まないものとする。
また、記憶容量で使用される補助単位を大きい順に並べると<span style="border:1px solid #000; padding: 0 1em 0 1em ">6</span> となり、1T(テラ)バイトは、1M(メガ)バイトの<span style="border:1px solid #000; padding: 0 1em 0 1em ">7</span> 倍である。
