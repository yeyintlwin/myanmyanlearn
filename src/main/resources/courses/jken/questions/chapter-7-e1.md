# 第7章 解説1

| No.   | Term (用語)              | Definition (解説)                                                                                       |
| :---- | :----------------------- | :------------------------------------------------------------------------------------------------------ |
| **1** | **なりすまし**           | 不正に入手した ID とパスワードを利用して、その人のふりをして正規のユーザであるかのように振る舞うこと。  |
| **2** | **フィッシング**         | 金融機関の Web サイトなどに似せたページに誘導して、個人情報を不正に入手すること。                       |
| **3** | **コンピュータウイルス** | 他人のコンピュータに入り込み被害を与えることを目的として作られたプログラム。                            |
| **4** | **スパムメール**         | 受信者の意思に関係なく大量に送信されてくる電子メール。広告や宣伝に使われる場合が多い。                  |
| **5** | **バックドア**           | 正規の認証以外の経路を用いてシステムへ再侵入するために、侵入者が設置する隠れた「窓口」。                |
| **6** | **ワクチンソフト**       | コンピュータに被害を与える不正なプログラムを検出し、除去するためのソフトウェア。                        |
| **7** | **フィルタリングソフト** | 不適切なキーワードや有害な画像を含む Web サイトを、特定の条件で閲覧できないように制限するソフトウェア。 |
