# 第3章 解説6

| No.   | Term (項目)     | Definition (解説)                                                            |
| :---- | :-------------- | :--------------------------------------------------------------------------- |
| **1** | **ブラウザ**    | Web サイトを閲覧するために必要となるソフトウェア。                           |
| **2** | **URL**         | Web サイトの位置情報や使用するプロトコル（http など）を記述したアドレス。    |
| **3** | **HTTP**        | 一般的に Web サイトを閲覧する際に使用される通信プロトコル。                  |
| **4** | **TCP/IP**      | インターネット上で提供される様々なサービスの基盤となる通信プロトコルの総称。 |
| **5** | **ドメイン名**  | 数字の羅列である IP アドレスを、人間が理解しやすい文字列で表現したもの。     |
| **6** | **DNS サーバ**  | ドメイン名を IP アドレスに変換（名前解決）する役割を持つサーバ。             |
| **7** | **DHCP サーバ** | LAN 内の各機器に対して、IP アドレスを自動的に割り当てる機能を持つサーバ。    |
