# 第7章 解説4

| No.   | Term (項目)          | Definition (解説)                                                                                      |
| :---- | :------------------- | :----------------------------------------------------------------------------------------------------- |
| **1** | **著作物の定義**     | 思想または感情を創作的に表現したもので、文芸、学術、美術または音楽の範囲に属するもの。                 |
| **2** | **著作権の帰属**     | 自分で購入した雑誌であっても著者に著作権があるため、無断でブログに公開してはならない。                 |
| **3** | **個人情報の同意**   | 利用目的や問合せ先を明示して提出を受ければ同意とみなされる。例外的に生命・財産保護時は無同意提供も可。 |
| **4** | **モデル写真の利用** | 自社パンフレット用であっても、事務所等の権利があるため勝手に Web サイトへ転載してはならない。          |
| **5** | **不正アクセス**     | ブラウザに情報が残っていても、他人の ID・パスワードでログインする行為は不正アクセスとなる。            |
| **6** | **情報の公開**       | メンバー本人の同意を得ている場合であれば、情報を公開しても問題はない。                                 |
| **7** | **迷惑メール規制**   | 特定電子メール法により「オプトアウト」の遵守や送信者情報の表示、罰則などが義務付けられている。         |
