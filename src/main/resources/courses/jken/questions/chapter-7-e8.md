# 第7章 解説8

| No.   | Term (項目)          | Definition (解説)                                                                               |
| :---- | :------------------- | :---------------------------------------------------------------------------------------------- |
| **1** | **名簿作成と配布**   | 利用目的と問合せ先を明示し、役員の承認を得て作成された名簿は、会員に配布することができる。      |
| **2** | **第三者提供の例外** | 法令に基づく場合や、人の生命・財産を保護するために必要な場合は、本人の同意なく情報提供が可能。  |
| **3** | **目的外利用の禁止** | 商品配送など、特定の業務のために取得した個人情報を、当初の目的以外に利用してはならない。        |
| **4** | **BCC の利用**       | 一斉送信時、TO や CC はアドレスが全員に見えてしまう。BCC を使うことでアドレスを非公開にできる。 |
| **5** | **処方箋の取扱い**   | 処方箋も重要な個人情報である。紛失や破損、漏洩が起きないよう適切に保護・管理する必要がある。    |
| **6** | **勤怠情報の保護**   | 勤務状況も個人情報にあたる。本人の許可なく再就職先の人事担当者などに提供してはならない。        |
| **7** | **未成年の同意**     | 新聞掲載等で本人の許可が必要な際、本人が未成年の場合は保護者の同意を得る必要がある。            |
