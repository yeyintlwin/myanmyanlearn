# 第4章 解説1

| No.   | Term (用語)            | Definition (解説)                                                                                      |
| :---- | :--------------------- | :----------------------------------------------------------------------------------------------------- |
| **1** | **ホームページ**       | Webブラウザを起動した時に表示される画面。ブラウザの設定でユーザが任意に変更できる。                    |
| **2** | **ブックマーク**       | 何度も閲覧するWebページを登録する機能。URLを入力せずにページを閲覧できる（お気に入り）。               |
| **3** | **ディレクトリ型検索** | 検索エンジンの一種。分野別の分類（カテゴリー）から目的のWebサイトを探す方式。                          |
| **4** | **ロボット型検索**     | 検索エンジンの一種。キーワードで絞り込みながらWebサイトを検索する方式。                                |
| **5** | **URL**                | インターネット上の情報にアクセスするための手順と所在を記述する規格。ホームページアドレスとも呼ばれる。 |
| **6** | **AND検索**            | 指定したキーワード全てを含んだWebページを検索する方式。                                                |
| **7** | **OR検索**             | 指定したキーワードのどれか一つでも含んだWebページを検索する方式。                                      |
