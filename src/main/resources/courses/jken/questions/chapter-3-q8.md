---

- <p>家庭内や会社内などの限られた範囲のネットワークを <span style="border:1px solid #000; padding: 0 1em 0 1em ">1</span> と呼ぶ。</p>
- <p>このネットワーク内のコンピュータ同士を接続する方法には、無線と有線がある。有線で接続する場合、各コンピュータから出るケーブルを集線するための装置である <span style="border:1px solid #000; padding: 0 1em 0 1em ">2</span> を利用する。また、インターネットなど外部ネットワークと接続する場合、<span style="border:1px solid #000; padding: 0 1em 0 1em ">3</span> と呼ぶ装置が必要となる。この <span style="border:1px solid #000; padding: 0 1em 0 1em ">3</span> から外部の交換器と接続する回線も技術の進歩とともに変化している。</p>
- <p><span style="border:1px solid #000; padding: 0 1em 0 1em ">4</span> は、ディジタル回線を用いて、電話、FAX、データ通信など複数の通信を統合している。</p>
- <p><span style="border:1px solid #000; padding: 0 1em 0 1em ">5</span> は、回線に従来の電話回線を利用し、ブロードバンド技術を利用して高速化を実現した。ただし、交換機までの上りと交換機からの下りで通信速度が異なる特徴がある。</p>
- <p><span style="border:1px solid #000; padding: 0 1em 0 1em ">6</span> は、光ファイバ回線を利用して高速に通信を行う。</p>
- <p><span style="border:1px solid #000; padding: 0 1em 0 1em ">7</span> は、ケーブルテレビの回線を利用して高速に通信を行う。</p>
