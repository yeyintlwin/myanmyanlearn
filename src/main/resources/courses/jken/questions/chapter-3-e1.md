| No.   | Term (項目)     | Definition (解説)                                                            |
| :---- | :-------------- | :--------------------------------------------------------------------------- |
| **1** | **LAN**         | 建物内や敷地内など、限られた範囲内で構築されるネットワークのこと。           |
| **2** | **WAN**         | 遠隔地のパソコンや離れた場所にある LAN 同士を接続した広域ネットワーク。      |
| **3** | **ISDN**        | 電話・FAX・データ通信を統合し、一つのデジタル回線で提供するサービス。        |
| **4** | **ADSL**        | アナログ電話回線で高速通信を行う技術。上りと下りの通信速度が異なるのが特徴。 |
| **5** | **CATV**        | ケーブルテレビの空き帯域を利用してインターネット接続などを提供するサービス。 |
| **6** | **FTTH**        | 光ファイバ回線を直接家庭などに引き込んで利用する高速通信サービス。           |
| **7** | **IP アドレス** | ネットワーク上の機器を一台ずつ識別するために割り当てられる固有のアドレス。   |
