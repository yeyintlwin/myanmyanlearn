# JKEN Chapter 2 – Question 3 Explanation

問2-3

（情報活用試験 3級 平成28年度後期問題2）

【解説】

1、2：複数の項目（ボタン）の中から一つだけを選択するのがラジオボタン、該当するものをすべて選択するのがチェックボックスである。

3：通常は表示されていないが、マウスを操作し必要な時に複数の項目（リスト）が表示されるのがドロップダウンリストである。そのリストの中から選択する。

4、5：メニューバーなどに並んでいるメニューを選択したときに、さらに詳細なメニューがすだ

れのように表示されるものがプルダウンメニュー、表示画面上の特定の位置で右クリックなとにより、メニューを浮かび上がらせて表示するものがポップアップメニューである。

6、7：年齢や性別、文化、障がいの有無や能力の違いにかかわらず、誰もが快適に利用できることをめざすデザインは、ユニバーサルデザインである。利用者の感じる使いやすさであるユーザビリティに配慮する必要がある。
