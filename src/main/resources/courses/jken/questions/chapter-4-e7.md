# 第4章 解説7

| No.   | Term (用語)        | Definition (解説)                                                                                 |
| :---- | :----------------- | :------------------------------------------------------------------------------------------------ |
| **1** | **MP3**            | 音楽CDと同等の音質を保ちつつ、音楽配信やプレーヤで広く利用されている代表的な音声ファイル形式。    |
| **2** | **MPEG**           | 動画の圧縮形式。ビデオ品質のMPEG-1や、ハイビジョン品質のMPEG-2など、用途に応じた規格がある。      |
| **3** | **SWF**            | Flashで作成されたムービー用のファイル形式。Webサイトの演出やゲームなどに使用される。              |
| **4** | **GIF**            | 最大256色を表示可能。透過処理やアニメーションに対応しており、ロゴや簡単な動画像に適している。     |
| **5** | **ストリーミング** | 動画や音声データを受信しながら同時に再生する技術。待ち時間が少なく、リアルタイム再生が可能。      |
| **6** | **PHP**            | HTML埋め込み型のスクリプト言語。データベースとの連携に優れ、動的なWebサイトの構築によく使われる。 |
| **7** | **CGI**            | Webサーバが外部プログラムと連携するための仕組み。掲示板やフォームなどで内容を動的に変化させる。   |
