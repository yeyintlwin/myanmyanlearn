## ＜設問 1 ＞

次のインターネットの仕組みに関する記述中の[]に入れるべき適切な字句を解答群から選べ。

---

<p>インターネット上のコンピュータや機器には、一台一台を識別する IP アドレスが付けられ、宛先アドレスなどとして使用している。現在は 32 ビットを 8 ビットずつ四つに区切り、それぞれを 10 進数で表し、192.168.60.180 のように表記される<span style="border:1px solid #000; padding: 0 1em 0 1em ">1</span>が主流である。後継の仕様として、128 ビットを用いるものも規格化されている。しかし、IP アドレスは人間にとって覚えやすいものではなく、「jken. sgec. or. jp」のように、わかりやすい<span style="border:1px solid #000; padding: 0 1em 0 1em ">2</span>を使用する。そのため <span style="border:1px solid #000; padding: 0 1em 0 1em ">2</span>を IP アドレスに変換する必要があり、この変換をしてくれるシステムが<span style="border:1px solid #000; padding: 0 1em 0 1em ">3</span>である。</p>

<p>また、機器一台一台に対して、IP アドレスを自動的に設定してくれるシステムが <span style="border:1px solid #000; padding: 0 1em 0 1em ">4</span> である。</p>
