| **2** | **パスワード管理**     | 有名なショッピングサイトであっても漏えいのリスクはある。なりすまし防止のため、定期的に変更する必要がある。   |
| **3** | **個人情報の収集**     | 利用目的の明示が必要。記載があったとしても、個人情報のむやみな入力は控えるように心がける必要がある。         |
| **4** | **プライバシー保護**   | たとえ友人であっても、自分のプライバシーを他人に教えたり見せたりしてはならない。                             |
| **5** | **不正通販サイト**     | 不正なサイトでの購入はしない。クレジットカード番号などの個人情報を搾取される危険性にも注意が必要。           |
| **6** | **ウイルス感染経路**   | ネットワーク経由だけでなく、記録メディアからも広がる。切断中も防げないため、対策ソフトを解除してはならない。 |
| **7** | **フィッシングメール** | 重要性や緊急性を装う文言を信じてはならない。原則として、不審なメールは開かないことが重要。                   |
