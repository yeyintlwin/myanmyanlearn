問 4-1 次の Web ブラウザの利用に関する記述を読み、各設問に答えよ。

---

Web サイトを閲覧するためには、Web ブラウザと呼ばれる専用のソフトウェアを使用する。Web ブラウザを立ち上げたとき最初に表示される Web ページは、<span style="border:1px solid #000; padding: 0 1em 0 1em ">1</span>と呼ばれ、ポータルサイトを設定する場合が多い。

閲覧したい Web ページの<u>(a) URL</u> を知っている場合は、直接 URL を入力して Web ページを開く。何度も閲覧する Web ページは、<span style="border:1px solid #000; padding: 0 1em 0 1em ">2</span>に登録しておくことにより、毎回 URL を入力することとなく閲覧することができる。URL が分からない場合でも、<span style="border:1px solid #000; padding: 0 1em 0 1em ">3</span>を利用することにより、目的の Web ページにたどり着くことができる。<span style="border:1px solid #000; padding: 0 1em 0 1em ">3</span>には、分野別の分類から目的の Web サイトを探すものや、<u>(b) キーワード</u> で絞り込みながら検索するものがある。前者は <span style="border:1px solid #000; padding: 0 1em 0 1em ">4</span> と呼ばれ、登録申請があった Web サイトの中で審査を通ったものがデータベースに登録されており、この中から検索される。
＜設問 1 ＞ 記述中のに入れるべき適切な字句を解答群から選べ。

＜設問 3 ＞下線部<u>(b) キーワード</u> で絞り込みに関する次の記述中の 1 に入れるべき適切な字句を解答群から選べ。
キーワード検索には、キーワード全てを含んだ Web ページを検索する AND 検索、キーワードのどれかを含んだ Web ページを検索する OR 検索がある。ここでは、AND 検索はキーワードを空白文字で区切り、OR 検索はキーワードを「OR」で区切って指定する。
例えば、次の A ～ D のような文章のページがあった場合、キーワードを「夏旅」とした場合は <span style="border:1px solid #000; padding: 0 1em 0 1em ">6</span> が、「夏 OR 旅」とした場合は <span style="border:1px solid #000; padding: 0 1em 0 1em ">7</span> がヒットする。

A：今年の夏休みは、北海道へ旅行に行きます

B：今年の夏休みは、北海道へ行きます。

C：今年の休みは、北海道へ旅行に行きます。

D：今年の休みは、北海道へ行きます。
