| No.   | Term (項目)        | Definition (解説)                                                                  |
| :---- | :----------------- | :--------------------------------------------------------------------------------- |
| **1** | **Web ページ**     | Web を構成する個々の文書のこと。                                                   |
| **2** | **HTML**           | Web ページを記述する言語。テキスト、写真、動画、リンクなどを組み込むことができる。 |
| **3** | **ブラウザ**       | Web ページを閲覧するために必要となるソフトウェア。                                 |
| **4** | **カテゴリ検索**   | 階層構造（大・中・小分類）を辿って目的の Web サイトを探す検索方法。                |
| **5** | **キーワード検索** | 特定の文字列（キーワード）を入力して目的の情報を絞り込む検索方法。                 |
| **6** | **ドメイン名**     | 数字列の IP アドレスを、人間が覚えやすい文字列（企業名など）に変換したもの。       |
| **7** | **DNS**            | ドメイン名と IP アドレスの対応関係を管理し、相互に変換を行うシステム。             |
