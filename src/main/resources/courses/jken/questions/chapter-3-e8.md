| No.   | Term (項目)    | Definition (解説)                                                                  |
| :---- | :------------- | :--------------------------------------------------------------------------------- |
| **1** | **LAN**        | 家庭内や企業内など、限られた範囲内で構築されるネットワーク。                       |
| **2** | **ハブ (HUB)** | 有線 LAN において、複数のコンピュータを物理的に集約して接続するための装置。        |
| **3** | **ルータ**     | インターネットなどの外部ネットワークと接続し、データの経路選択を行う装置。         |
| **4** | **ISDN**       | 電話・FAX・データ通信などの異なる網を統合し、デジタル回線で提供するサービス。      |
| **5** | **ADSL**       | 電話回線を利用した高速通信技術。上りと下りの通信速度が異なるのが大きな特徴。       |
| **6** | **FTTH**       | 光ファイバケーブルを家庭まで直接引き込み、超高速な通信を実現するネットワーク。     |
| **7** | **CATV**       | ケーブルテレビ網の空き帯域を利用して、インターネット接続サービスを提供する仕組み。 |
