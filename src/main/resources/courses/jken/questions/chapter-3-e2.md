| No.   | Term (項目)          | Definition (解説)                                                                                |
| :---- | :------------------- | :----------------------------------------------------------------------------------------------- |
| **1** | **NIC**              | ネットワークに接続するための拡張カードや内蔵回路。LAN ケーブルの接続と通信制御を行う。           |
| **2** | **MAC アドレス**     | NIC に割り当てられた 48 ビットの固有番号。物理層・データリンク層で機器の識別に使われる。         |
| **3** | **IP アドレス**      | ネットワーク層で設定するアドレス。IPv4 は 32 ビット、拡張された IPv6 は 128 ビットで構成される。 |
| **4** | **リピータハブ**     | 届いたデータをすべてのポートに中継する集線装置。一般的に「ハブ」と呼ばれる。                     |
| **5** | **スイッチングハブ** | MAC アドレスを判断して特定の相手にのみデータを送る機能（L2 スイッチ）を持つ集線装置。            |
| **6** | **ルータ**           | 異なる LAN 同士を接続し、IP アドレスを基に最適な経路を選択（ルーティング）する装置。             |
| **7** | **アクセスポイント** | 無線 LAN において、複数の端末からの通信を中継する基地局となる装置。                              |
