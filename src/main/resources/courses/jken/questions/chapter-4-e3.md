# 第4章 解説3

| No.   | Term (用語)            | Definition (解説)                                                                                        |
| :---- | :--------------------- | :------------------------------------------------------------------------------------------------------- |
| **1** | **ISP**                | インターネットサービスプロバイダ。電子メールやネット利用のために契約する接続サービス業者。               |
| **2** | **SMTPサーバ**         | 電子メールを送信（または転送）するためのサーバ。                                                         |
| **3** | **POPサーバ**          | 電子メールを受信するためのサーバ。メールサーバから自分のパソコンにメールを取り込む際に使用する。         |
| **4** | **署名（シグニチャ）** | メールの末尾に挿入する、送信者の氏名、会社名、連絡先などの情報。自動挿入の設定が可能。                   |
| **5** | **CC**                 | 「カーボンコピー」の略。メインの宛先以外の人にも参考に送る際に使用し、受信者全員にアドレスが表示される。 |
| **6** | **BCC**                | 「ブラインドカーボンコピー」の略。他の受信者にアドレスを知られずに、同じメールを送信できる機能。         |
| **7** | **Re:**                | 「Reply」の略。受け取ったメールに返信をする際、件名の先頭に自動的に付加される記号。                      |
| **8** | **Fw:**                | 「Forward」の略。受け取ったメールを別の人に転送する際、件名の先頭に自動的に付加される記号。              |
