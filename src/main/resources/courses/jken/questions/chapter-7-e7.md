# JKEN Chapter 7 – Online Ethics and Personal Safety

| No.   | Term (項目)        | Definition (解説)                                                                                  |
| :---- | :----------------- | :------------------------------------------------------------------------------------------------- |
| **1** | **不正通販サイト** | 著作権侵害サイトでの購入は厳禁。カード番号などの個人情報を搾取されるリスクが非常に高い。           |
| **2** | **個人情報の定義** | 生存する個人を識別できる情報のこと。行事予定や男女別入会人数などは個人情報には当たらない。         |
| **3** | **チェーンメール** | 内容が善意であっても、不特定多数への拡散を招くため、他の人に送らず自分のところで止める。           |
| **4** | **ブログへの掲載** | 自分の住所等は載せないのが原則。他人の情報を載せる場合は、必ず本人の許可が必要である。             |
| **5** | **不審メール対策** | フィッシングメールを疑い、安易に信じない。添付ファイルはウイルススキャン機能を有効にして確認する。 |
| **6** | **モバイルマナー** | 歩きスマホは衝突の危険がある。禁止場所では電源を切り、公共の場ではマナーモードで通話を控える。     |
| **7** | **写真の著作権**   | 写真には撮影者の著作権がある。友人が撮影した写真を SNS に載せる際は、撮影した友人の許可が必要。    |
