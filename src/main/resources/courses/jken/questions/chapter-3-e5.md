| No.   | Term (項目)     | Definition (解説)                                                                      |
| :---- | :-------------- | :------------------------------------------------------------------------------------- |
| **1** | **プロトコル**  | 通信を正しく行うための、送信側と受信側の間における手順や構造などの共通の決まり事。     |
| **2** | **TCP/IP**      | インターネット上で標準的に利用されているプロトコルの総称。                             |
| **3** | **ISP**         | インターネットへの接続サービスを提供する事業者（プロバイダ）。                         |
| **4** | **IP アドレス** | 機器の位置情報を表す 32 ビットの番号。8 ビットごとに 10 進数で表記しピリオドで区切る。 |
| **5** | **ドメイン名**  | 数字の羅列である IP アドレスを、人間が覚えやすい文字列（企業名など）で表現したもの。   |
| **6** | **DNS**         | ドメイン名をコンピュータが処理できる IP アドレスに変換（解決）する仕組み。             |
| **7** | **DHCP**        | ネットワークに接続された各機器に対し、IP アドレスを自動的に割り当てる仕組み。          |
