# 第4章 解説8

| No.   | Term (用語)        | Definition (解説)                                                                                          |
| :---- | :----------------- | :--------------------------------------------------------------------------------------------------------- |
| **1** | **URL**            | Webページのインターネット上の住所にあたるもの。Webブラウザに指定することで目的のページを閲覧できる。       |
| **2** | **サーチエンジン** | 目的のWebページのURLが分からない場合に、情報を探し出すために利用する検索サイト。                           |
| **3** | **ディレクトリ型** | 人の手で収集された情報を、あらかじめ用意された分野別のカテゴリ（分類）から辿って探す検索方式。             |
| **4** | **ロボット型**     | クローラというソフトが巡回して自動収集したデータから検索する方式。キーワード検索などで利用される。         |
| **5** | **SEO**            | 検索エンジン最適化。特定のWebサイトを検索結果のより上位に表示させるように工夫する技術のこと。              |
| **6** | **AND検索**        | 指定した全てのキーワードを含むページを検索する。絞り込みたい時に有効（例：AかつB）。                       |
| **7** | **OR検索**         | 指定したキーワードのうち、少なくともどれか一つを含むページを検索する。幅広く探す時に有効（例：AまたはB）。 |
