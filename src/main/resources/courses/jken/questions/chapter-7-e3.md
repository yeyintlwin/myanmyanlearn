# 第7章 解説3

| No.   | Term (項目)          | Definition (解説)                                                                    |
| :---- | :------------------- | :----------------------------------------------------------------------------------- |
| **1** | **著作権の発生**     | 著作権は、届け出をしなくとも創作された時点から創作した人に権利が発生する。           |
| **2** | **著作権の保護期間** | 2018 年改正で原則 70 年となった。出題時は個人・法人は 50 年、映画は 70 年であった。  |
| **3** | **特許権**           | 自然法則を利用した技術的思想の創作のうち高度なもの（発明）を保護する権利。           |
| **4** | **著作隣接権**       | 実演家や放送事業者など、著作物を公衆に伝達する役割を担う人に与えられる権利。         |
| **5** | **肖像権（選手）**   | 有名なスポーツ選手であっても、許可なくホームページに載せる行為は肖像権の侵害になる。 |
| **6** | **個人情報の権利**   | 本人が自分に関するデータの所在を確認し、訂正や削除を求める権利が保証されている。     |
| **7** | **個人情報の定義**   | 生存する個人を識別できる情報のこと。男女別卒業人数などは個人情報に当たらない。       |
