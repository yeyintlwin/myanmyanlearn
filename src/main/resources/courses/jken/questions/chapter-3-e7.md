# 第3章 解説7

| No.   | Term (項目)    | Definition (解説)                                                                          |
| :---- | :------------- | :----------------------------------------------------------------------------------------- |
| **1** | **ISP**        | インターネット接続サービスを提供する事業者（プロバイダ）。ISP を経由してネットに接続する。 |
| **2** | **ブラウザ**   | Web サイトを閲覧するために必要となるソフトウェア。                                         |
| **3** | **URL**        | Web サイトの位置情報や使用するプロトコル（http など）を記述したアドレス。                  |
| **4** | **HTTP**       | 一般的な Web サイトの閲覧に使用される通信プロトコル。                                      |
| **5** | **ドメイン名** | URL に記述される Web サイトの位置情報。ホスト名（サーバ名）と組み合わせて場所を特定する。  |
| **6** | **FTP**        | サーバに対してファイルのアップロードやダウンロードを行う際に使用するプロトコル。           |
| **7** | **TCP/IP**     | ネットワーク通信の規約（プロトコル）の総称。インターネットの標準規格。                     |
