# 第4章 解説2

| No.   | Term (用語)        | Definition (解説)                                                                                |
| :---- | :----------------- | :----------------------------------------------------------------------------------------------- |
| **1** | **ISP**            | インターネットサービスプロバイダ。個人がインターネットを利用する際に契約する接続サービス業者。   |
| **2** | **Webブラウザ**    | Webページを閲覧するために必要なソフトウェア。                                                    |
| **3** | **ac / ed**        | URLにおける組織の種類の識別子。acやedは大学などの教育機関を表す。                                |
| **4** | **jp / uk**        | URLにおける国や地域の識別子。jpは日本、ukはイギリスを表す。                                      |
| **5** | **org / net**      | URLにおける分類別の識別子。orgは非営利組織、netは通信組織（ネットワーク事業者）を表す。          |
| **6** | **カテゴリ検索**   | サーチエンジン（検索サイト）の方式の一つ。分野別の分類から目的のWebサイトを探す方式。            |
| **7** | **キーワード検索** | サーチエンジン（検索サイト）の方式の一つ。特定の単語（キーワード）を絞り込みながら検索する方式。 |
