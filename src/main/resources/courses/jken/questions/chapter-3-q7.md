- <p>世界中のネットワークが相互に接続され、巨大なネットワークとして張り巡らされたものをインすることができるシステムがWWWである。コンピュータをインターネットに接続するサービズを提供しているのが<span style="border:1px solid #000; padding: 0 1em 0 1em ">1</span>である。</p>
- <p>WWWのサービスを利用すると、インターネットに公開されているWebサイトを閲覧することができる。閲覧するためには<span style="border:1px solid #000; padding: 0 1em 0 1em ">2</span>と呼ばれるソフトウェアを利用し、<span style="border:1px solid #000; padding: 0 1em 0 1em ">3</span> を入力することで、画面にWebサイトのページが表示される。<span style="border:1px solid #000; padding: 0 1em 0 1em ">3</span>は、図に表すようにWebサイトの位置情報や、その時に使用するプロトコルを記述する。Webベージの閲覧で、一般的に使用するプロトコルが、<span style="border:1px solid #000; padding: 0 1em 0 1em ">4</span> である。</p>

図のところ

- <p>図の具体例において、ドメイン名を表している部分は <span style="border:1px solid #000; padding: 0 1em 0 1em ">5</span> である。また、Web サイトの閲覧以外にも様々なサービスが提供されており、ファイルのダウンロードやアップロードを行うプロトコルは <span style="border:1px solid #000; padding: 0 1em 0 1em ">6</span> である。である。このよこのような、インターネットを利用したネットワークサービスの規約はづいている。<span style="border:1px solid #000; padding: 0 1em 0 1em ">7</span> に基づいている。</p>
