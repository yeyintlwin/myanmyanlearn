# 第7章 解説6

| No.   | Term (項目)              | Definition (解説)                                                              |
| :---- | :----------------------- | :----------------------------------------------------------------------------- |
| **1** | **スパムメール**         | 受信者の意向に関わらず、営利目的で無差別に送られてくる大量の電子メールのこと。 |
| **2** | **オプトイン**           | 受信者が事前に送信者に対して、メール送信の同意を与えたり依頼したりすること。   |
| **3** | **フィッシング**         | 金融機関などを装い、口座番号や暗証番号などの個人情報をだましとる手口。         |
| **4** | **オプトアウト**         | 受信者が事前に送信者に対して、メールの受信拒否を通知すること。                 |
| **5** | **フィルタリングソフト** | 有害・違法なサイトへの接続を遮断し、閲覧を制限するためのソフトウェア。         |
| **6** | **ワクチンソフト**       | ウイルスの検知・駆除を行うソフト。最新の状態を保つため情報の更新が必須である。 |
| **7** | **電子透かし**           | 不正コピー検知のため、データ内に識別情報を埋め込み所有者を特定する技術。       |
