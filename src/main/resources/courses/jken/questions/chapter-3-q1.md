- <p>ネットワークの形態は、その規模により <span style="border:1px solid #000; padding: 0 1em 0 1em ">1</span> と<span style="border:1px solid #000; padding: 0 1em 0 1em ">2</span>に大別される。<span style="border:1px solid #000; padding: 0 1em 0 1em ">1</span>は、同じ建物や敷地内で通信を行う小規模なネットワークであり、<span style="border:1px solid #000; padding: 0 1em 0 1em ">2</span>は、遠隔地の<span style="border:1px solid #000; padding: 0 1em 0 1em ">1</span>同士を接続して構築された広域のネットワークである。遠隔地のコンピュータとネットワーク接続するには、利用する回線や通信の仕組みにより、次のような種類がある。</p>

- <span style="border:1px solid #000; padding: 0 1em 0 1em ">3</span>は、電話網とコンピュータネットワークを統合したディジタル回線網である。
- <span style="border:1px solid #000; padding: 0 1em 0 1em ">4</span>は、アナログの電話網にブロードバンド技術を適用して高速通信を行うネットワークであり、上りと下りで通信速度が異なる特徴がある。
- <span style="border:1px solid #000; padding: 0 1em 0 1em ">5</span>は、家庭などにひかれているケーブルテレビの回線を利用したネットワークである。
- <span style="border:1px solid #000; padding: 0 1em 0 1em ">6</span>は、光ファイバ回線を利用したネットワークである。

- <p>世界規模でコンピュータを相互に接続した巨大なコンピュータネットワークがインターネットである。インターネット上のコンピュータや機器には、<span style="border:1px solid #000; padding: 0 1em 0 1em ">7</span>と呼ばれる住所のような識別番号が付けられる。<span style="border:1px solid #000; padding: 0 1em 0 1em ">1</span>同士を接続する装置としてルータがある。ルータには複数のネットワークが接続されているが、ルータはこの<span style="border:1px solid #000; padding: 0 1em 0 1em ">7</span>を判断して、接続されているどの経路（ネットワーク）ヘデータを中継するかを決めている。</p>
