# 問6-4 次の情報化社会とコンピュータに関する記述を読み、関係の深い字句を解答群から選べ

1．自動車専用の有料道路を利用するときに、料金所などに設置された装置と自動車に搭載した数
置の間で通信を行い、自動車を停止させることなく、料金を自動収受するシステムである。

2．ナビゲーションシステムの位置測定に利用されている全地球測位システムである。最低3個の衛星からの電波を受信すれば、2次の位置が測定できる。

3.コンビニやスーパーのレジに置かれた端末装置で商品に付けられたバーコードを読み取り、店舗のレジで精算が行われた時点で商品管理や在庫管理を行うシステムである。

4.地域の商店街のように、複数の電子商店が出店しているWebサイトである。商品の受渡しはも配などを利用し、代金の支払いはクレジットカードやコンビニ払いなどを利用する。

5.インターネットとコンピュータを利用して、自宅などで好きな時間に学習できるシステムである。企業内研修で利用する場合は、業務システムと連携できるメリットがある。

6.様々なデータを格納した数ミリメートル程度のチップと、データを送受信するためのアンテナを内蔵しているものである。これが付けられた物品の商品情報や生産情報などを効果的に管理することができる。

7．コミュニケーションツールを提供するWebサイトの総称である。基本的な機能として、写真の公開やメッセージ交換などがある。また、個人に限らず、企業や自治体なども利用して情報発言を行っている。
