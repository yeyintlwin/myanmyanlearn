# 第3章 解説4

| No.   | Term (項目)          | Definition (解説)                                                                       |
| :---- | :------------------- | :-------------------------------------------------------------------------------------- |
| **1** | **IP アドレス**      | 機器を識別する 32 ビットの番号。8 ビットずつ区切り、10 進数をピリオドで繋いで表現する。 |
| **2** | **ドメイン名**       | 数字の羅列である IP アドレスを、人間が覚えやすい文字列（企業名など）で表現したもの。    |
| **3** | **DNS**              | ドメイン名をコンピュータが理解できる IP アドレスに変換する仕組み。                      |
| **4** | **DHCP**             | ネットワークに接続する各機器に対して、IP アドレスを自動的に割り当てる仕組み。           |
| **5** | **NIC**              | ネットワーク接続用の拡張カードや内蔵回路。LAN ケーブルの接続と通信制御を担う。          |
| **6** | **リピータハブ**     | 到着したすべてのパケットを中継・増幅して伝達する集線装置。増幅器とも呼ばれる。          |
| **7** | **アクセスポイント** | 無線 LAN において、複数の端末からのパケットを受け取り中継する基地局。                   |
