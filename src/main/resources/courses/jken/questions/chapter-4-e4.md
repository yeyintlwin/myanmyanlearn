# 第4章 解説4

| No.   | Term (用語)        | Definition (解説)                                                                                         |
| :---- | :----------------- | :-------------------------------------------------------------------------------------------------------- |
| **1** | **Webブラウザ**    | 目的のサーバからテキストや画像などをダウンロードし、パソコン上に表示・再生するためのソフトウェア。        |
| **2** | **ブックマーク**   | お気に入り機能。よく見るWebページを登録することで、URL入力の手間を省き素早く閲覧できる。                  |
| **3** | **https**          | SSL/TLSによる暗号化通信を行うプロトコル。情報の盗聴を防ぎ、安全にデータを送受信できる。                   |
| **4** | **or.jp**          | 日本国内に住所がある法人組織（一般社団法人や財団法人など）を表すドメイン名。                              |
| **5** | **ac.jp / ne.jp**  | ac.jpは日本の大学などの教育機関、ne.jpは日本のネットワーク事業者を指すドメイン名。                        |
| **6** | **サーチエンジン** | URLが不明な際、カテゴリ検索やキーワード検索を用いて目的のWebサイトを探すためのシステム。                  |
| **7** | **パンくずリスト** | Webサイト内での現在地を階層構造で示すナビゲーション。童話「ヘンゼルとグレーテル」のエピソードに由来する。 |
