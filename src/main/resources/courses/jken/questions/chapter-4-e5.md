# 第4章 解説5

![local_image](../images/c4q5_fig1.jpg)

| No.   | Term (用語) | Definition (解説)                                                                                    |
| :---- | :---------- | :--------------------------------------------------------------------------------------------------- |
| **3** | **GIF**     | 最大256色まで表示可能な形式。画像の輪郭をなじませる透過処理やアニメーションに対応している。          |
| **4** | **JPEG**    | 写真などの多色画像に適した形式。高い圧縮率を誇るが、元の画質に戻せない「非可逆圧縮」を採用している。 |
| **5** | **PNG**     | 写真などの多色画像を扱え、元の状態に戻せる「可逆圧縮」の形式。透過処理も可能。                       |
| **6** | **MPEG**    | 動画の圧縮規格。MPEG-1（ビデオ品質）やMPEG-2（ハイビジョン品質）など、品質に応じた複数の規格がある。 |
| **7** | **MP3**     | インターネットでの配信や音楽プレーヤーで広く利用されている、代表的な音声ファイル形式。               |
