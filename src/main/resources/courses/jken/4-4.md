# 4-4 　 WWW の利用（1）

Web サイトを利用するときの仕組みやブラウザの機能について学習します。

---

> WWW の仕組み

## (1) Web サイト

ている Web サイトを閲覧することができます。ブラウザで Web サイトを表示する仕組みを、以下に示します。
① クライアントから、見たい Web サイトの URL を指定する。
② URL に示されているドメイン名を頼りに、目的のサーバにたどり着く。
③ HTTP のプロトコルの取り決めにより、見たいページのデータをサーバからクライアントに送信する。
④ クライアント側のブラウザにより、画面上に Web サイトとして表示される。

## (2) URL

URL は、次のような規則によって記述されます。
スキーム名：／／ホスト（含むドメイン名）／パス名＜例＞

`http://www.jmam.co.jp/index.html`

---

> ブラウザの利用

Web サイトを閲覧するためには、ブラウザと呼ばれる専用のソフトウェアを使用します。ブラウザの代表的なソフトウェアとして、Internet Explorer や Firefox、Edge、Chrome などがあります。
これらブラウザには、ユーザが閲覧しやすいような機能が多く搭載されています。これらの代表的な機能には、次のようなものがあります。

## (1) URL の入力

閲覧したいホームページの URL を知っている場合、直接 URL を入力して Web サイトを開きます。

## (2) 検索

検索ボタンをクリックする。。キーワードの入力を行うだけしで、関連した Web サイトが検索できる機能です。
インターネット上にある膨大な情報の中から、必要な情報を探すための方法として検索エンジン（検索サイト）が利用されています。調べたい事柄についてキーワードを利用する方法と、あらかじめ分類されたカテゴリを利用する方法があります。
（ディレクトリ型検索エンジンでは、人が Web サイトを見てデータベースに登録していますロボット型検索エンジンでは、ます。定期的にインターネット上をクローラと呼ばれるプログラムが巡回して情報を収集し、データベースに登録していますています。

## (3) 更新

更新ボタンを押すと、表示されている URL の最新の情報が表示されます。

## (4) お気に入りの登録

何度も閲覧する Web サイトは、URL を登録することができます。登録しておくと、URL を入力することなく、Web サイトを閲覧することができます。
