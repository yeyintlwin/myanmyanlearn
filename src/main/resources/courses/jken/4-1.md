# 4-1　インターネットのサービス
インターネットの上に接続すると受けられる、一般的なサービスについて学習します。どのようなことができるのか、実体験もしながら理解しましょう。
>● インターネットのサービス

インターネットに接続すると、次のようなサービスを受けることができます。

## (1)  電子メール

**メーラ（メールソフト）** を使用し、送りたい相手のメールアドレスを指定することで、文章やデータ、画像などを相手に送ることができます。また、ブラウザを利用した**Webメール**のサービスもあります。

> ➤電子メールはEメールともいいます。

## (2) WWW
インターネットを通じて情報を共有したり、提供したりすることができるシステムです。Webサーバを用いてWebサイトを公開し、ブラウザを使ってWebサイトを閲覧します。
単に情報を見るだけでなく、**Java**や**CGI**などを使用して、**ブログ**や電子掲示板、チャットなど、Webサイト上に情報を発信していくことができます。
また、**電子商取引**などでは、インターネット上を流れるパスワードなどの盗聴を防ぐために、大切な情報を**暗号化**する技術が用いられています。
## (3) FTP
インターネット上に接続されたコンピュータ間で、ファイル転送を行うことができるシステムです。Webサイトに公開されているソフトウェアなどをダウンロードする場合にも、このサービスを使用します。
また、自分で作成したWebサイトをインターネット上に公開する場合も、このサービスを使用し、自分のコンピュータからWebサーバへファイルを転送（アップロード）します。

> ➤Java （ジャパ）：米国のSun　Microsystems社（現オラクル社）が開発したインターネット用のブログラミング言語です。4-5を参照

> ➤CGI (Common Gateway Interface)：4-5を参照

> ➤FTP: File Transfer Protocol
---
## (4) インターネット電話(IP電話)
インターネットを利用してリアルタイムの音声通信話を行うサービスです。また、**メッセンジャー（メッセージ送信ソフト）** では、文字、音声、映像をリアルタイムで送受信できるものがあります。
>● インターネットで利用される主なサーバ

インターネット上でサービスを受けるためには、さまざまなサーバを利用します。

| サーバ名              | 説明 |
|-----------------------|------|
| メールサーバ          | 電子メールの送受信を行う。 |
| Webサーバ             | Webサイトを公開しているサーバで，情報の公開や発信を行う。ユーザはブラウザでコンテンツを閲覧する。WWWサーバともいう。 |
| DNSサーバ             | ドメイン名からIPアドレスを検索して取得する。 |
| FTPサーバ             | ファイルの転送を行う。サーバに置かれているファイルの送受信を行う。 |
| ストリーミングサーバ | 映像や音楽などのマルチメディアデータの配信を行う。リアルタイム配信，オンデマンド配信が可能である。 |
---
> ➤ストリーミング（streaming）：映像や音楽などの大容量ファイルをダウンロードしながら再生する技術です。
