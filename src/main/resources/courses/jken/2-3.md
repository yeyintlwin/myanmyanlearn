# 2-3 　パソコンの発展と性能

コンピュータの歴史、およびパソコンの発展の概要と性能の動向について学習します。

---

> コンピュータの歴史

## (1) コンピュータの誕生

世界で最初に一般公開されたコンピュータは、アメリカ·ペンシルバニア大学の ENIAC（エニアック）で、1946 年のことでした。その後、コンピュータは急速に進歩して、1970 年代中ごろにパソコン（パーソナルコンピュータ）が登場しました。以後も発展を続けています。

## (2) パソコンの誕生

パソコン登場前後から現在までの歴史を簡単にたどると、次のようになります。

| 時代                | パソコンのテーマ | 説明（出来事）                                                                                                                     |
| :------------------ | :--------------- | :--------------------------------------------------------------------------------------------------------------------------------- |
| 1970 年代中ごろまで | 汎用機の時代     | コンピュータは高価なため、専用の部屋に置かれ、専門の操作員（オペレータ）のみが操作していた。                                       |
| 1990 年             | パソコンの発展期 | 日本 IBM が PC/AT 互換機で動作する、日本語対応 OS「DOS/V」を発表。                                                                 |
| 1995 年             | パソコンの浸透   | Windows95 の発売で、パソコンのデフォルトスタンダードは、Windows 搭載の PC/AT 互換機になった。                                      |
| 2000 年代           | パソコンの小型化 | デスクトップパソコンよりも、ノートパソコン、ネットブックと呼ばれる小型の PC がデフォルトスタンダードになった。                     |
| 2010 年代           | パソコンの多様化 | タッチディスプレイを搭載したタブレット PC や 3D ゲームを行うためのゲーミング PC など用途に合わせた様々な種類のパソコンが登場した。 |

---

> パソコンの発展

パソコン本体の高性能化と低価格化は、とどまるところを知らないかのように進んでいます。
下表は、おおよその年とその時代に主流であったパソコンの性能をまとめたものです。
性能は飛躍的に向上し、価格は反対に低下しているのがわかります。

| 性能/年                 | 1995 年ごろ | 2010 年ごろ | 2019 年 |
| :---------------------- | :---------- | :---------- | :------ |
| CPU 動作周波数（MHz）   | 100~        | 2,000~      | 3,000~  |
| コア数                  | 1           | 2           | 4       |
| メインメモリサイズ(MB)  | 16~         | 4,000       | 44,000  |
| ハードディスクサイズ(B) | 500M        | 250G~ 1T    | 4T      |
| 実売価格（万円）        | 30~         | 15 前後     | 10 前後 |

---

> 性能表の見方

次の表は、コンピュータのカタログに書かれている性能表のうち、おもな項目ついての説明です。コンピュータを購入するときは、使用方法と予算に合ったものを選ぶようにしましょう。

| 項目                        | 最近の値                                | 解説                                                                                                                |
| :-------------------------- | :-------------------------------------- | :------------------------------------------------------------------------------------------------------------------ |
| CPU 動作周波数(GHz)         | 1~5                                     | 1 秒間に実行するプログラムの命令回数を表します。数値が大きいほど高性能です。                                        |
| メインメモリサイズ(GB)      | 2G~128G                                 | プログラムが実行するときに使用できる広さを表します。数値が大きいほど高性能です。                                    |
| キャッシュメモリサイズ(MB)  | 2~16                                    | メインメモリより高速に動作するメモリです。数値が大きいほど高性能です。                                              |
| ハードディスクサイズ(TB/GB) | 32G~4T                                  | 作成したデータやファイルなどを保管します。容量は小さくなりますが、アクセス速度の速い SSD が使われることもあります。 |
| ネットワーク（通信速度）    | 4.8Gbps（無線）,1Gbps（有線）           | 1 秒間に何ビット送受信できるかを表します。数値が大きいほど高速にデータの送受信ができます。                          |
| CD-ROM（性能）              | 48 倍速（読み込み）,48 倍速（書き込み） | CD-ROM の読み込みや CD-R への書き込みの性能を表します。数値が大きいほど短時間に読み書きができます。                 |
| DVD-ROM                     | 24 倍速（読み込み）                     |                                                                                                                     |
| DVD-R                       | 24 倍速（書き込み）                     |                                                                                                                     |
| BD-ROM                      | 12 倍速（読み込み）                     |                                                                                                                     |
| BD-R                        | 16 倍速（書き込み）                     |                                                                                                                     |
