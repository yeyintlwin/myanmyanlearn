# 5-3 　アプリケーションソフトの基礎知識

アプリケーションソフトの種類やソフトウェアのインストールについてアプリケーションソフトの種類やソフトウェアのインストールに学習します。

---

> ソフトウェアの種類

コンピュータ上で起動するソフトウェアは、基本ソフトウェアとアプリケーションソフトウェアの 2 つに大別されます。ここでは、アプリケーションソフトについて述べます。アプリケーションソフトは、応用ソフトともいいます

|                        |                                               |
| :--------------------- | :-------------------------------------------- |
| 基本ソフトウェア       | OS のこと、コンピュータシステム全体を管理する |
| アプリケーションソフト | 特定の業務や目的のために使用するソフトウェア  |

---

> アプリケーションソフトの種類

## (1) 一般のアプリケーションソフト

一般のアプリケーションソフトは、ワード、表計算、ペイントなどの目的別に、次のように分けられます。
① ワープロソフト
② 表計算ソフト
③ データベースソフト
④ 画像ソフト
⑤ プレゼンテーションソフト
⑥ メールソフト
⑦ ブラウザ
現在、さまざまなソフトウェアが開発され、市販されています。したがって、アプリケーションソフトを購入する場合は、まず何をしたいかを考え、そのためのソフトウェアを選んでから、購入する必要があります。

## (2) 専用のアプリケーションソフト

市販されているソフトウェアでは、どうしても間に合わない業務もあります。たとえば、特殊なフォーマットの帳票出力が必要な場合や、独自の業務（航空会社の予約業務など）に必要なアプリケーションなどがこれにあたります。
これらの業務を行うためのソフトウェアは、別途それ専用に開発する必要があります。こうした場合は、ソフトウェアのオーダーメイドによって、開発を行っていきます。

---

> アプリケーションソフトのインストール

アプリケーションソフトを使用する場合、そのソフトウェアを購入後、コンピュータにインストールしなければ使用できません。
インストール作業を行うときに知っておかなければならないことは、アプリケーションソフトはどのコンピュータにでもインストールできるわけではないということです。
そもそもアプリケーションソフトは、どの OS で起動させるかを特定して、開発されています。また、ハードウェアの性能も加味したうえで、開発されています。これらの点に気をつけて使用しなければなりません。
アプリケーションソフトをインストールするためには、コンピュータに OS がインストールされていることが前提条件となります。
